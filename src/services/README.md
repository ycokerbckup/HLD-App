# Services Layer

Each Google Sheet has its own service layer:

- `members.service.ts` - Members sheet operations
- `announcements.service.ts` - Announcements sheet operations
- `schedule.service.ts` - Schedule sheet operations
- `training.service.ts` - Training sheet operations
- `reports.service.ts` - Technical reports sheet operations
- `feedback.service.ts` - Feedback sheet operations

Services transform Google Sheets data into typed objects.
