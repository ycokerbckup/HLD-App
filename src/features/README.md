# Feature Modules

Feature-first architecture with independent modules:

- `auth/` - Authentication and authorization
- `dashboard/` - Dashboard and home page
- `members/` - Member directory and profiles
- `announcements/` - Announcements feed
- `schedule/` - Duty schedule
- `training/` - Training progress tracking
- `reports/` - Technical reports
- `feedback/` - Feedback system
- `resources/` - Resources library
- `admin/` - Admin panel
- `profile/` - User profile
- `settings/` - Application settings
